UI Checklist — Demobaze

1. Nawigacja
- [ ] Menu główne działa poprawnie
- [ ] Logo przenosi na stronę główną
- [ ] Linki otwierają właściwe podstrony
- [ ] Brak błędów 404

2. Formularze (Login/Signup/Checkout)
- [ ] Prawidłowa walidacja pól
- [ ] Komunikaty o błędach są widoczne i czytelne
- [ ] Formularze można wysłać klawiszem Enter
- [ ] Wymagane pola są oznaczone

3. Produkty
- [ ] Zdjęcia produktów ładują się poprawnie
- [ ] Informacje o produkcie są czytelne (nazwa, cena, opis)
- [ ] Filtry działają poprawnie
- [ ] Sortowanie działa poprawnie

4. Koszyk
- [ ] Dodanie do koszyka aktualizuje licznik
- [ ] Można zmieniać ilość
- [ ] Można usuwać produkty
- [ ] Ceny i suma liczą się poprawnie

5. Responsywność
- [ ] Strona dobrze wygląda na telefonie
- [ ] Elementy nie nachodzą na siebie
- [ ] Menu mobilne działa poprawnie

6. Wygląd i czytelność
- [ ] Tekst jest czytelny
- [ ] Kolorystyka jest spójna
- [ ] Brak zepsutych obrazków
